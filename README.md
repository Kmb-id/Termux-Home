# STEP 1
- Install package git
  - ```pkg install git -y```
    
# STEP 2
```
git clone https://github.com/Kmb-id/Termux-Home
```
```
cd Termux-Home
```
```
chmod +x Auto.sh
```
# STEP 3 (Running)
```
bash Auto.sh 
```
