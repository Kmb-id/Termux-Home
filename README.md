- Merubah tampilan termux secara Automatis
  (setiap buka termux bannernya berubah auto berbeda-beda)
```
git clone https://github.com/Kmb-id/Termux-Home
```
```
cd Termux-Home
```
```
chmod +x Auto.sh
```
- Running 
```
bash Auto.sh 
```
